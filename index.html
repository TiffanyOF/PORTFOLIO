Site de video: SCARYTUBE

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma coleção de vídeos exclusivos de terror. O objetivo é criar um ambiente virtual onde seja possível compartilhar seus vídeos preferidos, fornecendo uma descrição breve de cada um, incluindo Scarytuber, o ano de publicação e uma opção de evento novo.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de vídeo em uma seção flexível (.vídeos). Cada vídeo é destacado em seu próprio contêiner (.vídeo), mostrando uma imagem da capa, descrição e um link de assitir. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores lembrando a Halloween e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração do terror.

Site interativo: Quente e Frio

O projeto consiste em uma aplicação web interativa, desenvolvida com HTML, CSS, e JavaScript, que simula uma experiência centrada no tema encontrar a bolinha. O objetivo é engajar os jogadores em testar seus limites para encontrar a bolinha flutuante.

O HTML estrutura o conteúdo da página, dividindo-o em seções como o Jogador e o Tesouro. O CSS é utilizado para estilizar a página, empregando um esquema de cores futurista e layout responsivo para melhorar a experiência do usuário. JavaScript é usado para adicionar interatividade ao jogo, manipulando o DOM para atualizar o conteúdo da página.

Os scripts JavaScript incluem módulos para gerar lugar aleatório, definir as o Jogador e as lógicas de jogo, como iniciar o jogo, processar as escolhas dos jogadores, e exibir o resultado final. Este design modular facilita a manutenção e a expansão do jogo. A importância das ferramentas utilizadas (HTML, CSS, JavaScript) reside na criação de uma experiência interativa.

Site de jogo: Ping-Pong

O projeto é uma aplicação interativa focada em interagir com um ou dois jogadores. Utiliza uma série de scripts associados a diferentes atores (sprites) e cenários para criar uma experiência de aprendizado envolvente. O objetivo é familiarizar os usuários com tópicos sobre jogos de mesa, através de interações e visualizações dinâmicas.

Cada ator no projeto possui scripts que definem seu lado em resposta, como o início do programa, mover a barra do usuário, ou pontos recebidas. Estes eventos desencadeiam ações como risadas, diversão e competividade sobre seus amigos . Por exemplo, a bolinha ir para as paredes, o usuário pode "receber" um ponto.

A importância das ferramentas utilizadas, como os blocos de código visual do Scratch, reside na sua acessibilidade e na capacidade de engajar aprendizes de diferentes idades no pensamento computacional e na exploração de conceitos gamers de forma interativa. O uso de cenários e movimentos ajuda a criar uma experiência mais rica e memorável, encorajando os usuários a explorar e interagir com o conteúdo de maneira ativa.
